# 个人博客主页 Nuxt3

#### 介绍

✨2024最新个人博客主页基于 Vue3、Nuxt3、Javascript、Pinia搭建 。可扩展公司官网、产品展示、在线简历...

![输入图片说明](docimage.png)




#### 环境

```shell
 Node.js >= 18.0.0

 #搭建时环境
 Node.js = 20.0.0
 npm = 10.2.0
 nuxt = 3.0.0-rc.1
 vue = 3.4.21
 
```
